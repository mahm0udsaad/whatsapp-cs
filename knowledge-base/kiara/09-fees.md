# الرسوم الإضافية (Fees)

## رسم وحيد مسجَّل: رسوم التوصيل

| البند | القيمة |
| --- | --- |
| اسم الرسم | رسوم التوصيل |
| المبلغ | 35 ر.س |
| نوع الحساب | مبلغ ثابت (Fixed) |
| متى يُطبَّق | على أي طلب يتطلّب وصول الأخصائية إلى موقع العميلة (الخدمة المنزلية) |
| شامل الضريبة؟ | نعم – المبلغ النهائي 35 ر.س (لا توجد ضريبة قيمة مضافة مفعّلة في كيارا حالياً) |
| هل هو خصم؟ | لا (رسم إضافي على الفاتورة) |

## ملاحظات

- لا توجد رسوم أخرى في النظام (لا رسوم حجز، لا رسوم خدمة، لا رسوم بطاقة).
- رسم التوصيل يُضاف **مرة واحدة لكل طلب** بغضّ النظر عن عدد الخدمات داخل الطلب.
- العميلات اللواتي يأتين إلى الفرع داخل الصالون لا تُحتسب عليهن رسوم التوصيل.
